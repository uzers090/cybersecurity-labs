# DNS TCP ICMP Analysis

## Objective
Analyze network traffic using Wireshark and identify common network protocols.

## Environment
- Kali Linux
- Oracle VM VirtualBox
- Wireshark

## DNS Analysis
Observed DNS queries and responses while accessing websites such as Google, GitHub, and TryHackMe.

## TCP Analysis
Observed the TCP three-way handshake:
- SYN
- SYN ACK
- ACK

## ICMP Analysis
Used ping to generate ICMP traffic and observed:
- Echo Request
- Echo Reply

## Tools Used
- Kali Linux
- Wireshark

- Packet Analysis
- Protocol Analysis
- Network Monitoring
- Troubleshooting
