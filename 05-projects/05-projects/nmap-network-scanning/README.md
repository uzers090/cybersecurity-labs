# Nmap Network Scanning Lab

## Objective
Learn host discovery, port scanning, and service detection using Nmap.

## Tools Used
- Kali Linux
- Nmap

## Activities
- Host Discovery
- Port Scanning
- Service Enumeration

## Findings

### Basic Scan
- Host localhost (127.0.0.1) was reachable.
- Nmap scanned 1000 TCP ports.
- No open TCP ports were detected.

### Service Detection Scan
- Performed service detection using Nmap.
- No active services were detected on localhost.

## Skills Learned
- Host Discovery
- Port Scanning
- Service Enumeration
- Nmap Usage
