# Home Cybersecurity Lab

## Objective
Build a personal cybersecurity lab for learning network security, Linux administration, virtualization, and security testing.

## Environment
- Kali Linux
- Ubuntu Server
- VMware / VirtualBox

## Activities
- Linux command practice
- Network analysis using Wireshark
- Basic security testing
- Virtual machine management
- Lab networking

## Skills Learned
- Linux
- Networking
- Virtualization
- Cybersecurity Fundamentals
- Troubleshooting

## Status
In Progress
